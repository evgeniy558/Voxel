
# Voxel
<img width="500" height="500" alt="Voxel icon" src="https://github.com/user-attachments/assets/14329702-c3ec-493e-9e44-f0c3fe5565fd" />
Voxel это музыкальное приложение с открытым исходным кодом, которое собирает треки из разных стриминговых сервисов в единую библиотеку. Voxel — единое пространство для вашей музыки.
