# Что это за приложение?
Voxel это музыкальное приложение для iOS с открытым исходным кодом, которое собирает треки из разных стриминговых сервисов в единую библиотеку. Приложение создано для людей, которые по разным причинам используют сразу несколько музыкальных платформ на своём смартфоне и хотят объединить их в одном удобном пространстве.
