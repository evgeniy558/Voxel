# Voxel music app for iOS
<img width="200" height="200" alt="Voxel icon" src="https://github.com/user-attachments/assets/4ede43bc-0e91-411f-854e-8a6465ede878" />
Voxel это музыкальное приложение для iOS с открытым исходным кодом, которое собирает треки из разных стриминговых сервисов в единую библиотеку. Voxel — единое пространство для вашей музыки.
